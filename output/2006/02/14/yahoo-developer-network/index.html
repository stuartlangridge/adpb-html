Yahoo Developer Network

Did I miss the bit where everyone else discovered the Yahoo Developer Network and forgot to tell me how much stuff there is there? They've BSD-licenced a shedload of JavaScript code as part of the Yahoo UI library, and there are some exceedingly useful design patterns: things like autocomplete textboxes or pagination or tabs and lots and lots of other bits. Normally I'd not be all that interested in things like this precisely because there are strings attached, but it's BSD licenced. That means we can all get ahold of the good stuff. Much kudos, Yahoo, I say.
