Using LinPhone on Ubuntu Breezy with SipGate

I have Voice over IP working! Using LinPhone on my Ubuntu 5.10 Breezy laptop, with SipGate UK as my provider. Instructions as follows:

  1. Sign up for a SipGate account.
  2. Once you've got one, sign into the SipGate website and click "My Account".
  3. Under "Connection Information", it will list your SIP-ID (a 7-digit number) and password, which you need for the next steps.
  4. Get LinPhone.
  5. Start it up, and say Go > Preferences.
  6. On the Network tab, set NAT traversal to "Enable", and enter your network's external IP address, which is the IP that you're connected to the net with. Find this out by going to What's My IP.
  7. Under RTP Properties, set "RTP port used for audio" to 5004.
  8. On the SIP tab, set "Run sip user agent on port" to 5060.
  9. Under Identity set your SIP address to sip:[username]@sipgate.co.uk, where [username] is your SIP-ID from step 3.
  10. Click "Add proxy/registrar". In the Proxy/Registrar configuration box, set Send Registration to on, Registration Period to 120, SIP Identity leave as "sip:", and "SIP Proxy" to "sip:sipgate.co.uk".
  11. Apply, Close.
  12. Quit LinPhone.
  13. On your firewall, forward these ports to your machine: 5060/udp, 5004/udp, 10000/udp, 8000-8012/udp, 3478-3497/udp, 3478-3497/tcp. (These might not all be required.)
  14. Start LinPhone again.
  15. You should have a "Proxy to use:" dropdown, which should be set to "sip:sipgate.co.uk".
  16. Test it all works by calling sip:613@fwd.pulver.com, which is an echo test.

All done. Hooray for VoIP!
